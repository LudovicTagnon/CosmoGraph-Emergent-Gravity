AUDIT CODEX INITIAL — ÉTAT DES LIEUX (IA1 → IA4)

Objet
Synthèse rapide de ce qui a été accompli dans le projet « Gravité émergente », avec les constats majeurs par phase et les pistes identifiées.

Phase IA1 / IA2 (champs gaussiens 3D, FMM)
- Modèles testés : baseline m ; K̃ = −Δ log c (FD7) ; K_Regge normalisé (5 tétras/cube, marge intérieure ≥2) ; fusion m+K̃+Regge.
- Résultats principaux (t=0, ROI 8–10³, L=18) : ΔrelRMSE ≈ −31 % pour K̃ ; ≈ −6–7 % pour Regge ; ΔPearson ≈ +2–3e−5. Gradient |∇m| rivalise avec K̃ ; combinaison gradient+K̃ marginalement meilleure.
- Robustesse temporelle (back-reaction rapide t=1–2) : la fusion m+K̃+Regge maintient un gain d’échelle (ΔrelRMSE ~ −2.6e−3) ; C-index se tasse mais reste ≥ 0. Marge Regge ≥2 impérative.

Phase IA3 (médiation statistique)
- Médiation t=0 validée avec instruments forts (F > 1900, ΔC-index +0.01–0.02). À t>0, la médiation échoue sous diffusion et instruments stricts : signal sensible au médiateur et à la dynamique du milieu.

Phase IA4 — piste topologique b0 (champs 1/f anisotropes L=32)
- Gains modestes mais significatifs : α=2.0 patch 7³ Δρ≈+0.0817, ΔrelRMSE≈−0.0065 ; α=1.5 patch 9³ Δρ≈+0.0567, ΔrelRMSE≈−0.0044 ; preuves de monotonie E[‖∇Φ‖|b0=k] et irréductibilité vs descripteurs lisses.
- Sandbox N-body : b0 quasi constant/inconsistant ; effet topologique faible → piste « b0 révolutionnaire » falsifiée ; forme/concentration explique mieux les résidus (compatible Poisson).

Analyses complémentaires (IA4, mode agent)
- Proxies locaux : combinaison (m, |∇m|², Δm) reproduit K̃ sans calcul intégral.
- Proxies entropiques : entropie locale 3×3×3 surpasse la courbure sur champs gaussiens (relRMSE 0.00525 → 0.00073) et améliore un champ sinusoïdal (~18 %). Courbure/géométrie nuisibles sur sinusoïdal ; aucun proxy utile sur champ aléatoire. Variance locale aide peu ; entropie+variance utile sur sinusoïdal.
- Diffusion t=0→2 : gains géométriques/entropiques persistent sur champs lisses/périodiques.

Constat global
- L’hypothèse « contenu → géométrie locale → cinématique » tient sur des champs lisses ; l’effet dépend du médiateur et de la dynamique du milieu.
- La piste topologique b0 est affaiblie/falsifiée hors champs synthétiques contrôlés.
- Les quantités entropiques locales émergent comme médiateurs plus généraux que la courbure seule dans certains cadres ; à explorer sur champs fractals/anisotropes et diffusions réalistes.
